# 　Pomo Frontend

ポモドーロ・タイマーを中心とした学習支援Webアプリのフロントエンドです。  
ユーザーはタイマー管理、タスク管理、学習ログの可視化を行うことができます。

## 🔗 デプロイURL
https://pomo-front.vercel.app

## 🛠 技術スタック
- React
- TypeScript
- Vite
- Firebase Authentication
- Axios
- CSS / HTML

## ✨ 主な機能
- Googleログイン / メールログイン（Firebase Auth）
- ポモドーロタイマー
- タイマーセットの作成・編集
- タスク管理
- 学習ログの記録
- 学習統計の可視化（今日・累計）
